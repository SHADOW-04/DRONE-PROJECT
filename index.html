Quick overview

Goal: Build a small, foldable carbon frame quad that is cheap, durable, and hackable. Flight controller is an Arduino Nano + MPU6050; controller is EvoFox One S (Bluetooth via HC-05 or 2.4GHz dongle). Use this repo to show your build process and firmware.

Wiring (summary)

Power → PDB → ESCs → motors. ESC signal wires to Arduino PWM pins. MPU6050 on I2C (SDA → A4, SCL → A5). HC-05 via SoftwareSerial or USB serial for pairing.

Battery (XT60) → PDB

ESC power → PDB | ESC signal → Arduino PWM pins MPU6050 → SDA(A4), SCL(A5), VCC, GND HC-05 → RX/TX (or SoftwareSerial pins) Keep common ground across all modules.

Firmware & example

This repo contains a minimal Arduino sketch that demonstrates mixer logic, ESC microsecond outputs and a skeleton for IMU reading. DO NOT fly without proper PID, arming, and safety checks.

Place full sketches in /firmware/arduino/ and document calibration steps in /docs/.

Build steps

  1. Assemble frame and mount motors.
  2. Solder ESC power to PDB and connect motors to ESCs.
  3. Mount Arduino + MPU6050 on vibration-isolated mount.
  4. Wire ESC signals to Arduino PWM pins; common ground.
  5. Connect HC-05 and pair with EvoFox (or use wired controller).
  6. Bench test with props OFF. Calibrate IMU, tune PID.

Safety

Always remove props while programming/tuning. Test in open areas and follow local rules. Add failsafe & arming sequence in firmware.
